Arbetsprocess

Jag började med att skapa en katalog på datorn med namnet dt211g-moment1 som var klonad till Git-hub. Därefter öppnade jag min terminal och placerade mig i den katalogen i terminalen och startade ett nytt npm-projekt genom kommandot "npm init". Sedan fyllde jag i de uppgifter jag tyckte var väsentliga och accepterade standardvärdena. Sedan installerades jag Parcel som ett utvecklingsberoende med kommandot "npm install parcel --save-dev". Därefter gjorde jag grundkonfigurationen i package.json.filen så att det framkommer vilken sida den ska starta på och vilka kommandon som ska gälla för att bygga webbplatsen. Genom att använda mig av Parcel så kommer mycket automatisering ske automatiskt när jag bygger webbplatsen. CSS kommer minifieras, transpilering och minifiering av Javascript kommer ske om jag anger vilka verisioner som det ska göras för, minifiering av HTML, kopiering va arbetsfiler till mapp för publicering samt optimering av bilder så länge man anger i koden vad som ska göras. Därefter skapade jag en mappstruktur för webbplatsen med en src-mapp och mappar för css, bilder och javaScript samt skapade alla html-sidor. Därefter uppdaterade jag html-sidorna, css och JavaScript med meny och enklare struktur för webbplatsen. För att se till så att allt fungerade som tänkt körde jag sedan npm start och öppnade url i terminalen i webbläsarfönstret för att se att allt fungerade som tänkt. Därefter kopplade jag repot till netlify och testade att publiceringen fungerade. Sedan la jag in alla bilder i HTML-koden i rätt fromat så att parcel sköter bildoptimeringen automatiskt.
